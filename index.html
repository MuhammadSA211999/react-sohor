Q.3: What does JavaScript Event Loop do?

JavaScript is a single thread, generic & event revent functional programming language. Usually Javascript code is executing synchronous-lly but any fuction calling like API fetch, timeout and setInterval are exicuting asychronouslly. In execution phase if V8 engine or Node JS found a asynchronous function or a million line code that need more time to execution are send to WEB API or C++ API for Node JS to proper execution. When Call Stack(Last in First Out) will empty, Event Loop call Queue(First in First Out) and call the WEB API and send the function to Call Stack(Last in First Out). Event Loop make a connection with Call Stack & Queue .Exactly, Javascript Event Loop prevent Executing Block Behevior to make JavaScript fast.
